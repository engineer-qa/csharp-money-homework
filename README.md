# Домашние задания по C#
Задание 1. Класс Money и Product

Реализован класс Money для работы с денежными значениями:

Инкапсулированы поля: целая часть и копейки (0–99).

Добавлена валидация и форматированный вывод (125.90 грн).

Реализован метод задания суммы с проверкой корректности данных.

На основе Money создан класс Product с полями:

Name, Price (Money)

Методы IncreasePrice() и DecreasePrice() корректно обновляют копейки и гривны.

Работа классов продемонстрирована в консольном меню.

Задание 2. ООП — Музыкальные инструменты

Создан базовый класс MusicalInstrument и производные классы:
Violin, Trombone, Ukulele, Cello.

Для каждого инструмента реализованы методы:

Show() — отображение названия

Desc() — описание характеристик

History() — краткая история происхождения

Использованы наследование, инкапсуляция и переопределение (override).
Реализован полиморфный вызов методов через коллекцию:

List<MusicalInstrument> orchestra = new();

Задание 3. Структура DecimalNumber

Создана структура DecimalNumber для работы с десятичными числами:

Содержит одно поле (или свойство) для целого числа.

Реализованы методы:

ToBinary()

ToOctal()

ToHex()
Каждый метод возвращает строку с результатом преобразования.

Результаты выводятся в консольном меню в удобном виде.
